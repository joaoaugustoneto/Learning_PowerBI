## 💻 Como Visualizar e Interagir com os Projetos

Para abrir, analisar os relacionamentos e testar as fórmulas DAX dos arquivos contidos na pasta `treinos/`:

1. Faça o download e instale o [Power BI Desktop](https://powerbi.microsoft.com/pt-br/desktop/) (Gratuito).
2. Clone este repositório ou baixe os arquivos desejados diretamente da pasta `treinos/`.
3. Abra o arquivo `.pbix` de sua preferência no Power BI Desktop.
